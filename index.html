⚠ 브라우저 CORS 제한 안내
브라우저에서 trino.opensky-network.org에 직접 접근하면 CORS 정책으로 차단됩니다.
권장: 로컬 Node.js 프록시를 실행하거나 데모 모드를 사용하세요.
비워두면 직접 연결 시도 (CORS 차단 가능)
Node.js 프록시 서버 설정 방법 ▸ 
npm install express http-proxy-middleware cors
// server.js
const express = require('express');
const cors = require('cors');
const { createProxyMiddleware } = require('http-proxy-middleware');

const app = express();

app.use(cors());

app.use('/trino-proxy', createProxyMiddleware({
  target: 'https://trino.opensky-network.org',
  changeOrigin: true,
  pathRewrite: { '^/trino-proxy': '' },
  secure: true
}));

app.listen(8080, () => {
  console.log('Trino proxy ready: http://localhost:8080/trino-proxy');
});
→ 프록시 URL 입력: http://localhost:8080/trino-proxy
